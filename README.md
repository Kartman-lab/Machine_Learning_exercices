# Machine Learning Exercises

Ce projet regroupe plusieurs exercices pratiques en Machine Learning visant à maîtriser les concepts fondamentaux tels que la régression linéaire, la régression logistique et le clustering. Chaque exercice est réalisé à partir de datasets réels et permet de mettre en pratique l’ensemble du workflow de Machine Learning : exploration des données, prétraitement, modélisation, évaluation et visualisation des résultats.

---

## Structure du projet

### 01_Regression_Lineaire
Exercices de régression linéaire pour prédire des valeurs continues à partir de différentes caractéristiques :

- **Medical_Cost_Personal_Dataset.ipynb** : Prédiction des charges médicales des individus selon l’âge, le sexe, l’IMC, le nombre d’enfants, le statut de fumeur et la région.
- **Student_Performance.ipynb** : Prédiction des notes finales d’étudiants en fonction de facteurs liés à l’étude et aux caractéristiques personnelles.

### 02_Regression_Logistique
Exercices de régression logistique pour la classification binaire :

- **Titanic_Survival.ipynb** : Prédiction de la survie des passagers du Titanic en fonction de caractéristiques personnelles.

### 03_Clustering
Exercices de clustering pour identifier des groupes similaires dans les données sans labels :

- **Mall_Customers.ipynb** : Segmentation de clients d’un centre commercial selon leur âge, revenu et habitudes de dépenses.

## Objectifs pédagogiques
- Maîtriser les concepts de base du Machine Learning (régression, classification, clustering).
- Comprendre le workflow complet d’un projet ML : exploration, prétraitement, modélisation, évaluation et visualisation.
- Expérimenter avec des datasets réels et interpréter les résultats.
- Préparer des projets pour un portfolio ou un futur projet plus avancé.
